Описание

Встраиваемая вытяжка CC4527S с 4 скоростями работы, позволяющими подобрать оптимальный режим для текущих условий.

Модель шириной 52 см — хороший выбор для малогабаритной кухни. Корпус целиком изготовлен из полированной нержавеющей стали. Две светодиодные лампы обеспечивают яркую подсветку рабочего места. На электронной панели управления с кнопками находится таймер, автоматически отключающий вытяжку через заданное время. Расположение мотора внутри вытяжки уменьшает уровень шума. Антивозвратный клапан не позволяет загрязненному воздуху и посторонним запахам из вентиляционной системы проникнуть в помещение. Четыре скорости всасывания рассчитаны на разный уровень загрязненности воздуха.

Вытяжка предназначается для работы в режиме отвода воздуха с подключением к системе вентиляции и в режиме рециркуляции с очисткой воздуха при помощи системы фильтров. Два жироулавливающих фильтра из нержавеющей стали пригодны для мытья в посудомоечной машине. О необходимости очистки оповещает специальный индикатор. Отдельно можно приобрести монтажный комплект для подключения работы в режиме рециркуляции и угольный фильтр для предохранения от неприятных запахов. Максимальная производительность 670 м3/ч позволяет быстро очистить воздух в помещении даже на низкой скорости.
